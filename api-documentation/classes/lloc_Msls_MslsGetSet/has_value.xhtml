lloc\Msls\MslsGetSet::has_value

Checks if the array has an non empty item with the specified key name.

This is method is similar to the overloaded __isset-method since __set cleans empty properties but I use for example $obj->has_value( $temp ) and not isset( $obj->$temp ) which is the same but in my opinion a little bit ugly.
